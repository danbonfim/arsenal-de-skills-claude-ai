# 💬 Guia de Prompts por Skill

Use este guia como referência rápida de **como ativar e interagir** com cada skill dentro do seu projeto no Claude AI.

---

## 💰 Alex Hormozi — The Offer Architect

> Especialista em estruturação de ofertas irresistíveis, precificação e value stacking.

```
"Audite minha oferta: [descreva seu produto/serviço, preço e público]"
"Qual seria a Grand Slam Offer para [seu nicho]?"
"Como estruturo o value stack para justificar [preço]?"
"Crie 3 opções de garantia para meu programa de [descrição]"
"Rode meu negócio pela Value Equation de Hormozi"
"Quais lead magnets funcionam melhor para [nicho]?"
```

---

## 📱 Gary Vaynerchuk — The Content Strategist

> Especialista em conteúdo, marca pessoal e crescimento orgânico nas redes sociais.

```
"Crie uma estratégia de conteúdo para [plataforma] focada em [nicho]"
"Monte um sistema de conteúdo pilar para mim. Meu tema principal é [tema]"
"Quais formatos de conteúdo tenho que priorizar como [cargo/papel]?"
"Crie um plano de 30 dias de conteúdo para o meu lançamento de [produto]"
"Como repurposo um artigo de blog para LinkedIn, Instagram e YouTube Shorts?"
```

---

## 🔁 Russell Brunson — The Funnel Architect

> Especialista em funis de vendas, email marketing e storytelling para conversão.

```
"Projete um funil completo para meu [produto/serviço]"
"Crie o script de Perfect Webinar para minha oferta de [descrição]"
"Construa o value ladder para meu negócio de [tipo de negócio]"
"Escreva uma sequência de 5 emails para leads que não compraram"
"Qual seria o Epiphany Bridge para minha história de origem?"
"Analise meu funil atual: [descreva cada etapa]"
```

## 🚀 Patrick Bet-David — Entrepreneur Strategist

## 💡 Dica Pro: Combine Skills

Você pode solicitar que o Claude use múltiplas perspectivas ao mesmo tempo:

```
"Com a visão do Hormozi na oferta e do Brunson no funil, crie minha
estratégia completa de lançamento para [produto]"

"Use Cialdini para o copy e Kennedy para o posicionamento e crie
um anuncio de vendas para [produto]"

"Com a mentalidade de Abraham em crescimento e Naval em leverage,
qual a melhor estratégia de expanso para meu negócio de [tipo]?"
```
